## 第四步：部署后端服务

### 4.1 选择部署平台

推荐平台（任选其一）：

| 平台 | 优点 | 免费额度 |
|------|------|----------|
| Railway | 简单易用 | 每月 $5 |
| Render | 自动部署 | 免费层可用 |
| 阿里云函数计算 | 国内快 | 有免费额度 |
| 腾讯云 Serverless | 国内快 | 有免费额度 |
| 自己的服务器 | 完全控制 | - |

### 4.2 以 Railway 为例部署

1. 打开 https://railway.app/ 并登录
2. 点击「New Project」→「Deploy from GitHub repo」
3. 选择你的仓库，设置根目录为 `server`
4. 添加环境变量：
   ```
   PORT=3000
   TT_APP_ID=tt4186373bdfe20c5401
   TT_APP_SECRET=9600b4cfcc9ccd95ac8417b4cf8b28db19de81fb
   ```
5. 等待部署完成，获得公网地址（如 `https://fanpick-server.up.railway.app`）

### 4.3 更新小程序 API 地址

修改 `miniapp/utils/api.js` 第 6 行：

```javascript
// 修改前
const BASE_URL = 'https://your-server.com/api';

// 修改后（替换为你的实际地址）
const BASE_URL = 'https://fanpick-server.up.railway.app/api';
```

## 第五步：配置服务器域名（重要！）

### 5.1 登录开放平台

1. 打开 https://developer.open-douyin.com/
2. 进入你的小程序应用

### 5.2 添加域名

1. 左侧菜单：「开发管理」→「开发设置」
2. 找到「服务器域名」部分
3. 点击「修改」，添加：

   | 类型 | 域名 |
   |------|------|
   | request 合法域名 | `https://fanpick-server.up.railway.app` |

4. 点击「保存」

> ⚠️ **注意**：
> - 域名必须是 **HTTPS**
> - 域名需要**备案**（国内服务器）
> - 每月最多修改 **5 次**

## 第九步：等待审核

### 9.1 审核时间

- 通常 **1-3 个工作日**
- 高峰期可能更久

### 9.2 审核状态

在「版本管理」页面查看：
- 🟡 审核中
- 🟢 审核通过
- 🔴 审核驳回（需根据驳回原因修改后重新提交）

### 9.3 常见驳回原因及解决

| 驳回原因 | 解决方案 |
|----------|----------|
| 服务类目不符 | 修改为「工具 → 信息查询」 |
| 缺少隐私协议 | 添加隐私协议页面 |
| 功能不完整 | 确保所有功能可用 |
| 内容违规 | 检查文案和图片 |

## 常见问题

### Q: 上传按钮灰色无法点击？

A: 检查是否已登录，以及 AppID 是否正确。

### Q: 预览时白屏？

A: 检查网络请求是否被拦截，确认服务器域名已配置。

### Q: 真机无法登录？

A: 确认 `TT_APP_SECRET` 配置正确，后端服务正常运行。

### Q: 审核一直不通过？

A: 查看驳回原因，常见问题是类目不符或缺少隐私协议。
